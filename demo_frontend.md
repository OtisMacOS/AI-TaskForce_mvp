# 🎨 AI Think Tank MVP 前端演示

## 🚀 快速启动演示

### 1. 启动前端服务器
```bash
cd frontend
python server.py
```

### 2. 启动后端API服务器（可选）
```bash
# 在项目根目录
source venv/bin/activate
cd ai_thinktank_mvp
uvicorn api.main:app --reload --host 0.0.0.0 --port 8000
```

### 3. 访问前端界面
打开浏览器访问: **http://localhost:3000**
